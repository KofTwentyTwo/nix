Standout Engineering Highlights

🛡️ Sandbox-Safe Secret Deployment

The custom mkPatDeployer block in your SOPS module solves a major pain-point with declarative environments. Standard sops-nix symlinks point into ~/.config/sops-nix/, which dangle inside sandboxed environments like Claude Cowork container folders. Your deployer decrypts secrets directly inside work directories using atomic shell operations (mktemp/mv) with absolute target permissions (0600).

🤖 Zero-Prompt AI Agent Orchestration

Your configuration is uniquely optimized to host local and cloud-based terminal coding agents:

  • Claude Code Permissions: Over 150 read-only, navigation, and local diagnostic tools are explicitly whitelisted in settings.local.json, removing user prompt fatigue for safe commands.
  • Pi TypeScript Extensions: Dynamic behavioral extensions (like safe-bash.ts) and custom markdown templates are maintained directly in the flake, propagating to all machines on switch.
  • Ollama Synchronization: Massive 70GB LLM models (e.g. qwen3-coder:30b, llama3.3) are pulled idempotently on system activation.

⌨️ Apple Silicon Sleep / USB Wake Patch

Apple Silicon's deep sleep states cut USB power, disabling onboard state registers and custom RGB lighting profiles on gaming keyboards (like the Razer Huntsman V3 Pro) since Razer Synapse is absent on macOS. You resolve this system-wide using an activation PMSET patch to maintain low-level USB power preservation.

💡 Step-by-Step Linux Enablement Strategy

To successfully integrate Linux without breaking your macOS fleet, execute the following strategy:

  • 1. Single-Entry Flake Strategy (Unified `flake.nix`): Instead of managing two disconnected files, unify your architectures inside the existing flake.nix. Add a homeConfigurations attribute set alongside your darwinConfigurations.
  • 2. Decouple Homebrew Dependency: Move shared CLI tools (like bat, eza, fzf, zoxide, ripgrep, jq) from homebrew.brews in modules/homebrew.nix into Home Manager's declarative package array inside home/default.nix.
  • 3. Abstraction of System Paths: Change hardcoded absolute paths to use Home Manager's interpolation helper: ${config.home.homeDirectory}.
  • 4. Clean Up $PATH Integrations: Wrap your Darwin-specific PATH additions conditionally so they compile cleanly on both architectures using lib.optionals pkgs.stdenv.isDarwin [ ... ].
