# Customer Intake Checklist

All fields marked **required** must be collected before creating records in Zoho or Confluence.
Fields marked *optional* should be asked about but can be skipped.

## Company Information

| Field | Required | Notes |
|-------|----------|-------|
| Company name | **Yes** | Official legal name |
| Industry / vertical | **Yes** | |
| Website | **Yes** | |
| Street address | **Yes** | |
| City | **Yes** | |
| State / Province | **Yes** | |
| Zip / Postal code | **Yes** | |
| Country | **Yes** | Default: US |

## Primary Contact

| Field | Required | Notes |
|-------|----------|-------|
| Full name | **Yes** | First and last |
| Title / role | **Yes** | |
| Email | **Yes** | |
| Phone | **Yes** | |
| Preferred contact method | *Optional* | Email, phone, text |

## Deal Information

| Field | Required | Notes |
|-------|----------|-------|
| Deal source | **Yes** | Referral, inbound, outbound, event, partner |
| Products / services of interest | **Yes** | |
| Estimated deal value | *Optional* | |
| Expected close date | *Optional* | |

## Zoho CRM Setup

| Field | Required | Notes |
|-------|----------|-------|
| Account type | **Yes** | Prospect, Customer, Partner |
| Owner / assigned rep | **Yes** | Defaults to current user if not specified |
| Tags | *Optional* | |

## Confluence Setup

| Field | Required | Notes |
|-------|----------|-------|
| Parent space | **Yes** | Which Confluence space to create under |
| Page template | *Optional* | Standard or custom; defaults to standard |
